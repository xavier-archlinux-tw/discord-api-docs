## Discord 中的多人體驗 (Multiplayer Experiences in Discord)

互動活動 (Activities) 是直接嵌入在 Discord 中的多人遊戲和社交體驗。它們可以在頻道、私訊 (DM) 中啟動，或者從應用程式啟動器 (App Launcher) 啟動，不需要外部視窗或單獨下載。

玩家可以與已經在語音頻道中的好友一起加入，這使得互動活動非常適合派對遊戲、共同觀看、協作工具等。

![Discord 互動活動](/images/platform/activity-playing.png)

## 用於互動活動的嵌入式應用程式 SDK (Embedded App SDK for Activities)

在底層，互動活動是一個透過 [嵌入式應用程式 SDK (Embedded App SDK)](/developers/zh-tw/developer-tools/embedded-app-sdk) 與 Discord 進行通訊的 iframe。該 SDK 提供了在您的互動活動與 Discord 用戶端之間進行互動的方法。
