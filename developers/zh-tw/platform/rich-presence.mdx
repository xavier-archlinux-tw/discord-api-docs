豐富狀態 (Rich Presence) 可讓您的遊戲在使用者啟用活動分享時，在其 Discord 個人資料上顯示即時、可操作的資料：例如他們正在玩什麼、處於哪一個關卡、在該活動階段中待了多久，或是提示好友加入的按鈕。這是一個輕量級的整合，可以將您的應用程式連結到 Discord 的社群功能層。

要呈現哪些資料完全由您決定。豐富狀態支援自訂文字欄位、時間戳記、隊伍大小指標以及上傳的插圖資產。配置「加入 (Join)」按鈕後，好友可以直接從個人資料卡片跳轉到使用者的遊戲活動階段。

豐富狀態可透過兩個 SDK 取得，具體取決於您要建置的項目：適用於原生遊戲整合的 Discord 社交 SDK (Discord Social SDK)，或者適用於在 Discord 內部運行之互動活動的嵌入式應用程式 SDK (Embedded App SDK)。

## 選擇 SDK (Choosing an SDK)

在整合豐富狀態時有兩種選擇。根據您要建置的項目，您需要為您的專案選擇正確的 SDK：

- **[Discord 社交 SDK (Discord Social SDK)](/developers/zh-tw/discord-social-sdk/overview)** 允許您在遊戲或應用程式中建立社群功能，包括好友列表、遊戲邀請等。
- **[嵌入式應用程式 SDK (Embedded App SDK)](/developers/zh-tw/developer-tools/embedded-app-sdk)** 應該在您於 Discord 中建置 [互動活動 (Activity)](/developers/zh-tw/activities/overview) 時使用。

<Info>
豐富狀態資料會公開顯示在您的 Discord 個人資料上，因此在開發期間，您應該使用僅屬於您私有開發伺服器的測試帳戶。
</Info>

### Discord 社交 SDK (Discord Social SDK)

Discord 社交 SDK 用於直接在您的遊戲中建置社群功能。當使用者正在玩您的遊戲時，您可以在他們的 Discord 個人資料中展示他們正在做什麼，甚至可以透過遊戲邀請提示他們的好友加入。

當將豐富狀態與平台外遊戲整合時，可以顯示有關使用者在您的遊戲中正在做什麼的資料。還可以配置「加入 (Join)」按鈕，以允許使用者的好友加入他們的遊戲並發送遊戲邀請。

[了解如何使用 Discord 社交 SDK 實作豐富狀態](/developers/zh-tw/discord-social-sdk/development-guides/setting-rich-presence)

### 嵌入式應用程式 SDK (Embedded App SDK)

嵌入式應用程式 SDK 用於建置互動活動 (Activities)，這些活動是託管在 Discord 內部的 iframe 中的多人遊戲和社交體驗。該 SDK 處理 Discord 與互動活動之間的通訊，並協助整合平台功能（例如豐富狀態！）。

在使用者加入互動活動後，豐富狀態可用於動態顯示該使用者正在做什麼或玩什麼的資料，並提示其他人加入並一起遊玩。

[了解如何使用嵌入式應用程式 SDK 實作豐富狀態](/developers/zh-tw/rich-presence/using-with-the-embedded-app-sdk)
