應用程式營利功能適用於 [Bot (機器人)](/developers/zh-tw/bots/overview) 以及 [互動活動 (Activities)](/developers/zh-tw/activities/overview)。如果您正在建置遊戲整合，請參閱 [社群商務與遊戲商店](/developers/zh-tw/social-commerce/overview) 以獲取更多關於如何使用 Discord 營利您遊戲的資訊。

我們的付費應用程式 (Premium Apps) 功能可讓您直接在 Discord 內部為您應用程式的功能進行收費，並使用 Discord 內建的付款與結帳流程。使用者無須離開平台即可進行訂閱或購買，這減少了摩擦力並保持了原生的體驗。Discord 處理帳單、詐騙檢測與收據，而您則處理授予存取付費功能的權限。

您可以提供兩種類型的營利方式：

- **循環按月訂閱 (Recurring monthly subscriptions)**：可選擇提供給個人或整個伺服器
- **單次購買 (One-Time purchases)**：可選擇提供永久性升級或消耗型道具

這兩種類型都透過同一個 [SKU](/developers/zh-tw/resources/sku) 和 [授權實體 (Entitlement)](/developers/zh-tw/resources/entitlement) API 進行整合，因此您應用程式可以在授予存取權限之前檢查使用者是否擁有該功能的存取授權。

已啟用營利功能的應用程式還可以在應用程式目錄上獲得額外的曝光版面，包括一個商店頁面，使用者可以在該頁面上瀏覽與購買您的產品，而無需先安裝您的應用程式。
