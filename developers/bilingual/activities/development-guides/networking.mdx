## Activity Proxy Considerations

All network traffic is routed through the Discord Proxy for various security reasons. WebRTC and WebTransport details are covered in the main guide.

## Construct A Full URL

```javascript
const protocol = `https`;
const clientId = '<YOUR CLIENT ID>';
const proxyDomain = 'discordsays.com';
const resourcePath = '/foo/bar.jpg';
const url = new URL(`${protocol}://${clientId}.${proxyDomain}${resourcePath}`);
