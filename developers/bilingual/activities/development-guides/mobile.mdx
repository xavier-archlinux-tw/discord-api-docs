## Supported Platforms: Web, iOS, Android

Enable/disable platform support in the [Activities -> Settings](https://discord.com/developers/applications/select/embedded/settings) tab of the developer portal.

## Mobile Safe Areas

```css
:root {
  --sait: var(--discord-safe-area-inset-top, env(safe-area-inset-top));
  --saib: var(--discord-safe-area-inset-bottom, env(safe-area-inset-bottom));
  --sail: var(--discord-safe-area-inset-left, env(safe-area-inset-left));
  --sair: var(--discord-safe-area-inset-right, env(safe-area-inset-right));
}
```

## Mobile Thermal States

Subscribe to thermal state changes:

```javascript
discordSdk.subscribe('THERMAL_STATE_UPDATE', handleThermalStateUpdate);
```

States: `NOMINAL(0)`, `FAIR(1)`, `SERIOUS(2)`, `CRITICAL(3)`

## 行動裝置安全區域 (Mobile Safe Areas)

```css
:root {
  --sait: var(--discord-safe-area-inset-top, env(safe-area-inset-top));
  --saib: var(--discord-safe-area-inset-bottom, env(safe-area-inset-bottom));

  --sair: var(--discord-safe-area-inset-right, env(safe-area-inset-right));
}
```

## 行動裝置散熱狀態 (Mobile Thermal States)

訂閱散熱狀態變更事件：

```javascript
discordSdk.subscribe('THERMAL_STATE_UPDATE', handleThermalStateUpdate);
```

狀態：`NOMINAL(0)`、`FAIR(1)`、`SERIOUS(2)`、`CRITICAL(3)`
