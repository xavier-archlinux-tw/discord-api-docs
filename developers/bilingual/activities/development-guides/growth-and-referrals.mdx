## Prompting Users to Share Incentivized Links

Use links for referrals, promotions, social deep-links, turn-based deep-links, affiliate marketing, and source attribution.

```javascript
const { success } = await discordSdk.commands.shareLink({
    message: 'Click this link to redeem 5 free coins!',
    custom_id: customId,
});
```

## Generating a Custom Link

```javascript
const linkIdResponse = await fetch(`${env.discordAPI}/applications/${env.applicationId}/quick-links/`, {
  method: 'POST',
  headers: { Authorization: `Bearer ${accessToken}`, 'Content-Type': 'application/json' },
  body: { custom_id: 'user_123/game_456', title: 'Check out my high score!', image }
});
const {link_id} = await linkIdResponse.json();
await discordSdk.commands.shareLink({ message: 'Check out my high score!', link_id });
