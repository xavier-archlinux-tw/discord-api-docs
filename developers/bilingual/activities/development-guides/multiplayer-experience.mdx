## Activity Instance Management

When a user clicks "Join Application", they expect to enter the same application that their friends are participating in. The `instanceId` is the key for your application to identify which unique instance it is talking to.

```javascript
import {DiscordSDK} from '@discord/embedded-app-sdk';
const discordSdk = new DiscordSDK(clientId);
const instanceId = discordSdk.instanceId; // available immediately
```

**Instance ID Semantics:** Generated when a user launches an application. All joining users receive the same `instanceId`. Ends when all users leave.

## 活動實例管理 (Activity Instance Management)

當使用者點擊「加入應用程式」時，他們期望進入與朋友相同的應用程式實例。`instanceId` 是識別特定實例不可或缺的依據。

```javascript
import {DiscordSDK} from '@discord/embedded-app-sdk';
const discordSdk = new DiscordSDK(clientId);
const instanceId = discordSdk.instanceId; // 立即可用
```

**instanceId 的語義**：在使用者啟動應用程式時生成。所有加入的使用者收到相同的 `instanceId`。所有使用者離開後，該實例即結束。

## Instance Participants

```javascript
// Fetch
const participants = await discordSdk.commands.getInstanceConnectedParticipants();

// Subscribe
discordSdk.subscribe(Events.ACTIVITY_INSTANCE_PARTICIPANTS_UPDATE, updateParticipants);
```

## Render Avatars and Names

```javascript
const {user} = await discordSdk.commands.authenticate({ access_token: accessToken });

let avatarSrc = user.avatar
  ? `https://cdn.discordapp.com/avatars/${user.id}/${user.avatar}.png?size=256`
  : `https://cdn.discordapp.com/embed/avatars/${(BigInt(user.id) >> 22n) % 6n}.png`;

const username = user.global_name ?? `${user.username}#${user.discriminator}`;
```
